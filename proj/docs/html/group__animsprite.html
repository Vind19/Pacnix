Detailed Description

Functions for animated sprites.

Animated Sprites are Sprites with a set of images available, and switch between them on a certain rate

Function Documentation

int animate_asprite ( AnimSprite *  fig)

Animate an Animated Sprite.

Animates a AnimSprite by incrementing the cur_aspeed counter and checking if the value os aspeed was reached. When this happens, it switches to the next image on the array of images

Parameters
figpointer to AnimSprite to animate
Returns
0 if animation occurred (image was switched), 1 otherwise
