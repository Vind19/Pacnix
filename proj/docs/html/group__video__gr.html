void update_buffer ( )

Updates the video_mem buffer with contents of double, secondary buffer.

This function is used to implement the double buffering feature of the program. When anything is written to the screen, it is actually written to a temporary buffer, video_mem_buffer. Upon passing of a new frame (controlled by function caller) The auxiliary buffer is coppied to the actual video_mem buffer, making it appear on the screen. This prevents problems on large information display with several sprites (moving or not)
