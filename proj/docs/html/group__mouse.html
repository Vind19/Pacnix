int mouse_read_packet ( Mouse_packet *  mouse)

Reads a 3 byte mouse packet upon separated interrupts.

Is called as a handler for mouse interrupts, modifying the Mouse_packet given when the third and last byte was read It ignores the first x bytes read (specified in a global variable) since first couple of packets are often incorrect

Parameters
mousepointer to Mouse_packet to modify
Returns
1 upon third and last byte, 0 otherwise
