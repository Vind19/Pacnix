Detailed Description

Struct representing a Bonus.

This struct represents the fruit bonuses that sometimes appear at the pacman's spawn place. Information about the score value of the bonuses, their active/inactive status, time to spawn and duration is stored here. A Bonus is only consumable when it is active. When it is active and is consumed, It becomes inactive and the spawn counter is reset
