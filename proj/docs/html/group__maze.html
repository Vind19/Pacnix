Detailed Description

Functions related to Pacman maze.

The game map is represented by a matrix of integer values. Each value of the matrix specifies the content of a 30*30px cell in the screen. The content can be either a path for the pacman and ghosts (empty path, cell with dot, cell with energizer) or a wall. There are several types of walls to make up the design of the map.
